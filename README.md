## 📌 Project Description
This project is a voice-enabled hospital appointment booking system.  
Users can interact using voice or text to book appointments.

## 🚀 Features
- 📞 Call simulation UI
- 🎙️ Voice input (Speech Recognition)
- 🔊 Voice output (Text-to-Speech)
- 🤖 Rasa chatbot integration
- 🏥 Appointment booking system

## 🛠️ Technologies Used
- HTML, CSS, JavaScript
- Rasa (Python)
- Web Speech API
